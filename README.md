# morphotrack

Reconstruct morphogenic tracks.
Relevant paper: 
> Murakami et al., 2022

### Software prerequisites 

Tested on Ubuntu 20.04 LST with the following versions of software.
- Cmake 3.21.3
- Python 3.8.12
- CUDA Toolkit 11.6
- [deform](https://github.com/simeks/deform)
- ISPC 1.16.1
